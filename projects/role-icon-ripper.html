role-icon-ripper

bulk-download every custom role icon from a discord guild

node cli discord api

screenshots

features

  • pulls every role icon from a guild in one shot
  • handles animated (gif) and static (png) icons
  • safe filename normalization
  • direct discord api calls, no discord.js overhead
  • drops everything into a single output directory

intended for

server owners backing up custom assets, or admins migrating between guilds who want to reuse their role icon set without re-uploading each one by hand.
