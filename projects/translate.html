translate

public translation bot with a right-click menu

discord.js public bot

screenshots

features

  • right-click context menu translation
  • per-user language preferences
  • multi-target translations in one call
  • auto language detection on source
  • preferences stored as comma-separated lang codes
  • public bot — add to any server

intended for

multi-language discord communities where server-wide auto-translation would be too noisy, but on-demand translation of individual messages via right-click is exactly the right amount of friction.
