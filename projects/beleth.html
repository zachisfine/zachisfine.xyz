beleth

parallel ssh brute-force for authorized security testing

python paramiko c libssh2

screenshots

features

  • parallel worker pool with configurable concurrency
  • dual implementations: python (paramiko) & c (libssh2)
  • wordlist-driven, standard dictionary format
  • banner grabbing for openssh version identification
  • rate and elapsed-time reporting per run
  • early-exit on first successful credential

intended for

authorized pentesters and ctf competitors verifying ssh hardening, and security researchers benchmarking password policy against realistic parallel attack patterns. not for use against systems you don't have written permission to test.
