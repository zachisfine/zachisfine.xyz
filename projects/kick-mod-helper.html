kick-mod-helper

floating moderator panel for kick.com

chrome extension vanilla js mv3

screenshots

features

  • floating overlay that runs on kick.com chat
  • spam detection with configurable rules
  • user watch list with quick-mute
  • keyword filters with whitelist support
  • chat-velocity monitoring
  • muted user review section
  • message fingerprinting to survive virtual-scroll reuse
  • chrome mv3 extension, no external services

intended for

kick.com streamers and moderators who need real-time visibility over chat during high-activity broadcasts — especially when raid traffic, coordinated spam, or new followers make manual moderation impractical.
