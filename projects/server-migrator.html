server-migrator

migrate messages between discord servers

node express discord.js

screenshots

features

  • discord-to-discord message migration
  • webhook replay preserves original author appearance
  • live sse dashboard: rate, eta, event stream
  • graceful stop-flag checks at multiple points
  • auto file-attachment handling past size limits
  • rate-limit-aware pacing
  • resumable across multiple runs

intended for

communities consolidating multiple servers into one, or moving content to a new home while keeping message history intact and authors looking right. built to be pausable and resumable across long-running migrations.
